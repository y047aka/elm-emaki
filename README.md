# Emaki

## Quick Start

### 1. Install `elm-emaki` cli & Emaki Package

```shell
$ npm install git@github.com:y047aka/elm-emaki.git#main
$ elm install y047aka/emaki
```

### 2. Write `Emaki.elm`
